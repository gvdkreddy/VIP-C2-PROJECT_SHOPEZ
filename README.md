# SHOPEZ

A full-stack MERN E-Commerce application built using:

- React.js
- Node.js
- Express.js
- MongoDB Atlas

## Features

- User Authentication
- Product Catalog
- Search & Filtering
- Shopping Cart
- Wishlist
- Checkout System
- Order Management
- Admin Panel
- Recently Viewed Products
- Coupon Discounts

## Installation

Frontend:
npm install
npm run dev

Backend:
npm install
npm run server
